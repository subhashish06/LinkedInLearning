# Find Prime Factors

This project is from the "Level Up Python by Barron Stone" course on LinkedIn Learning. It contains a Python script that calculates the unique prime factors of a given positive integer, along with a comprehensive suite of unit tests.

## Files

- `prime_factors.py`: The core module. It includes the `find_prime_factors(n)` function and an interactive command-line interface.
- `test_prime_factors.py`: Contains unit tests testing various edge cases such as negative numbers, zero, one, primes, and composite numbers with duplicate or distinct factors.

## How to Run

To use the tool interactively:

```bash
python prime_factors.py
```
*You will be prompted to enter a number, and the program will return the unique prime factors.*

To execute the test suite:

```bash
python test_prime_factors.py
```
